# Effective Mobile Sub Service API

## Стек
- Go
- Gin Framework
- PostgreSQL
- Docker Compose

**Все требования выполнены**

## Как запустить
1. Выполните `docker-compose up --build`.
2. API доступно на `http://localhost:8080/api`.
3. Swagger-документация на `http://localhost:8080/swagger/index.html`.
